# FnOS 飞牛OS dynv6 SSL 全自动续证一键脚本

**解决飞牛官方不支持 dynv6 自动SSL续期的痛点｜一键部署｜永久自动续证｜无需90天手动上传**

---

## 📖 项目简介

飞牛OS自带SSL自动续期仅支持：阿里云、腾讯云、华为云、Cloudflare 等主流服务商。

广大NAS用户常用的 **dynv6 免费DDNS域名** 官方无法自动续期，默认每90天需要手动申请、手动上传、手动替换证书，非常麻烦。

本脚本基于官方开源 **acme\.sh** 开发，专为飞牛OS适配，**一条命令全自动部署**，实现永久自动续期。

---

## ✅ 主要功能

- 自动安装/升级官方 acme\.sh 证书工具

- 自动申请 **主域名 \+ 泛域名 \*** ECC加密SSL证书

- 支持自定义证书导出目录，兼容所有飞牛挂载路径

- 导出完整4类证书文件，标注「必传/可选/禁止上传」

- **自动识别飞牛系统证书ID目录**（无需用户查找路径）

- 自动替换系统证书、自动更新数据库有效期时间戳

- 自动重启 Nginx/网关/WebDAV/SMB 服务，证书即时生效

- 系统每日自动检测，提前30天自动续签，永久无需手动操作

- 自动加固权限，保护Token与私钥安全

## 🧩 支持域名说明

- **✅ 完全支持：dynv6 所有免费域名**

- **❌ 不支持：FreeDNS 二级域名（bot\.nu / mooo\.com）**（服务商API限制无法自动TXT验证）

- 其他域名（Cloudflare/阿里云等）如需适配可联系作者获取多服务商版本

## 📝 部署流程（全程傻瓜式）

1. 执行一键脚本

2. 依次输入：你的 dynv6 域名、dynv6 Token、提醒邮箱

3. 证书保存路径默认即可（可自定义）

4. 脚本自动申请证书并导出文件

5. 根据提示，在飞牛后台手动上传一次证书

6. 上传完成回车，脚本自动完成后续所有部署

7. 从此永久自动续期，无需干预

## 📂 飞牛后台证书上传规范（新手必看）

脚本导出4个文件，**严格按以下规则上传**

- **✅ 证书文件（必传）：fullchain\.cer**

- **✅ 私钥文件（必传）：你的域名\.key**

- **🟡 中间证书（可选）：issuer\_certificate\.crt**（普通用户可留空）

- **❌ 禁止上传：域名\.crt**（裸证书、无证书链、会导致浏览器不安全）

**推荐最简方案：仅上传 fullchain\.cer \+ 私钥，中间证书留空**

## 🔄 自动续期机制

- 系统每日自动检测证书状态

- 证书剩余有效期 ≤30天 自动触发续签

- 自动签发90天新证书、自动覆盖系统路径

- 自动更新飞牛数据库过期时间

- 自动重启服务，无缝生效、不会断网

## 🧷 常用维护命令

模拟每日自动检测（排查故障用）

```Plain Text
acme.sh --cron
```

查看证书到期时间

```Plain Text
openssl x509 -in /usr/trim/var/trim_connect/ssls/你的域名/*/fullchain.crt -dates -noout
```

强制测试续期流程

```Plain Text
acme.sh --renew -d 你的域名 --force
```

## ❓ 常见问题

#### Q：普通用户执行提示 acme\.sh: command not found

A：正常现象！自动续期由 root 定时任务后台执行，不影响自动化。

#### Q：提示未找到飞牛证书目录？

A：必须先在飞牛网页后台「新增证书并保存」一次，才会生成系统目录。

#### Q：为什么不支持 freedns 二级域名？

A：服务商限制共享二级域名禁止API自动验证，属于官方限制无解。
